カラオケプレイヤー - 使い方
========================

■ 仕様
・歌詞は「全文表示」で、再生位置に合わせて自動スクロール
・MP3 と同名の .lrc を同じフォルダに置く（例: my-song.mp3 → my-song.lrc）

■ 使い方
1. index.html をブラウザで開く
2. 「フォルダを選択」で、MP3 と .lrc が入ったフォルダを選ぶ
3. 一覧から曲を選ぶと再生開始。歌詞が表示され、現在の行がハイライト＆スクロール

■ LRC 形式（1行の例）
[00:12.00] ここに歌詞
  → 分:秒.百分秒 で表示タイミングを指定

■ 注意
・ブラウザの都合で、フォルダは「選択したフォルダ内」のファイルだけ読みます
・スマホで車にミラーリングする場合は、このHTMLをスマホで開けるようにして使ってください
